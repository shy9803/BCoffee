#BCoffee

2024년에 신규로 만든 웹사이트 파일입니다.
주변에서 흔히 볼 수 있는 카페를 토대로 가상의 브랜드를 만들어 이를 웹사이트를 만들었습니다.
B는 Best라는 단어의 B이기도 하며, 벌 Bee를 의미하기도 하는 등 다양하게 연상할 수 있습니다.
벌은 고객을, 카페는 꽃을 의미하여 사람들이 자연스레 많이 방문하도록 하는 걸 상징하고자 비유적으로 생각했습니다.
